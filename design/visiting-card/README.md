# Visiting card — print instructions

**File:** `siri-vruddhi-visiting-card.html`

## How to use

1. Double-click `siri-vruddhi-visiting-card.html` (opens in Chrome or Edge).
2. Click **Save as PDF / Print**.
3. Enable **Background graphics** in the print dialog.
4. Choose **Save as PDF** — you get one PDF with **front** (page 1) and **back** (page 2).
